Product Pitch 01 / 05

DeskGuard
Library Space Management

A real-time, anti-hoarding web portal solving seat management friction in busy study environments. Eliminates seat reservation misuse through server-managed automated sweepers.

  • The Problem: Students reserve desks with backpacks and disappear for hours.
  • The Fix: Smart scanning, live maps, check-ins, and dynamic away timers.

Features Matrix 02 / 05

Frictionless Seat Occupancy & Flow

  • Student Reservation: Scan a printable desk label QR code to automatically check in and hold a desk.
  • Step Away Timer (20 mins): Pause sessions for coffee breaks. Automatically frees the desk if away timer runs out.
  • Active Session Controls: Students can check out instantly or declare returns.
  • Librarian Overrides: Dashboard controls allowing chief librarians to manually "Force Reset" occupied desks.

System Architecture 03 / 05

Under the Hood: Modern Tech Stack

  • Frontend Core: React (Vite) client providing a responsive layout, Zinc dark themes, and custom SVG floor plans.
  • Real-time Engine: Server-Sent Events (SSE) stream pool broadcasting live desk updates to all browsers in sub-seconds.
  • Security Layer: User registration and login powered by bcryptjs password hashing and JWT token verification.
  • Database: Supabase PostgreSQL for cloud persistence, with local JSON file fallback for developer utility.

Deployment & Operations 04 / 05

Operation & QR Label Print System

  • Automatic Print Formats: Dedicated @media print styling automatically hides dashboard widgets, layouts, and sidebars.
  • Sticker Sheets: Clean 2-column sticker grid matching standard A4 printer label sheets.
  • Dynamic QR Server: Integrated on-the-fly QR code generation linking to the frontend deployment.
  • One-Click Hosting: Deployed static assets on Vercel and persistent background task node processes on Render.

Value Proposition 05 / 05

Why DeskGuard Matters

  • Fair Seating Occupancy: Prompts open, fair study spacing allocation. Prevents seats remaining empty while booked.
  • Administrative Controls: Provides library staff with live usage states and force resets.
  • Frictionless Check-Ins: Scan-to-reserve eliminates physical badge check-in registers.
  • Extensible Framework: Future expansion support for seat maps editors, locking penalties, and peak charts.
